IoT based waste management system

This system is mainly developed for Dhaka city. It always seems that wastages cannot manage properly in Dhaka city. This system will help create smart IoT based trash bin that will send notification to the relevant authority to clean the trash when it become fill up. This system will also help to navigate the smaeter route via which the driver can find the traffic free road to carry the trash on their truck for throwing purpose.

Via this system the traveller can also get the notification of updated ticket price, seat availibility, distance between source and destination.This system has been developed by using Xampp software and also in the backend several database queries has been used.But at the front end user can easily control the system as a simple and interective UI has been used here
